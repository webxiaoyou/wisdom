## canvas手写板，超漂亮的横屏签名

下载下来后，将组件中f-signature.vue复制到您的组件中


### 使用方法

引用组件--注意您组件的路径
```
import fSignature from '@/components/module/f-signature/f-signature';
```

案例

```
<f-signature></f-signature>
```
